# GraphMind Frontend — Context Log
Last updated: 2026-02-20

## Project Status
All M3 (Frontend) tasks from Weeks 1–4 are implemented.

## Architecture
- **Framework:** React 19 + Vite 8 (SPA)
- **Styling:** Tailwind CSS v4 (via @tailwindcss/vite plugin)
- **State Management:** Zustand (single store: useAppStore.js)
- **Graph Visualization:** @xyflow/react (React Flow)
- **Icons:** lucide-react

## Data Layer
- **Mock data:** src/data/mockData.js (users, docs, mindmaps, chat responses, fallback)
- **Store:** src/store/useAppStore.js (Zustand — users, docs, mindmap, chat, ingest, UI state)
- **Backend calls are commented out** in the store; currently uses simulateChatResponse / simulateIngest.

## Features Implemented
1. Multi-user switcher (Alice Morgan / Bob Chen) with isolated memory spaces
2. Sidebar with collapsible nav, ingested documents list, ingest button
3. Chat window with markdown-bold rendering, typing indicator, auto-scroll
4. PerformanceTimer: "Retrieval completed in X ms" (excludes LLM time)
5. CitationBadge: clickable citations that navigate to the mindmap and highlight nodes
6. GraphCanvas: React Flow with 4 custom node types (concept, entity, document, fact), edge labels, animated edges, minimap, controls, node detail panel
7. IngestModal: title + content form with loading state and success feedback
8. Graceful fallback: irrelevant queries return "I don't know / not in memory" message
9. Error boundary for crash recovery
10. Dark theme with custom color palette, animations (fadeIn, slideIn, pulse glow, typing dots)
